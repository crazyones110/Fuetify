## Introduction

## Get Started

1. add Some CSS

  Before using this framework, please set `border-box`
  ```css
  *, *::before, *::after {
    box-sizing: border-box;
  }
  ```
  Support IE8+

2. install fuetify

  `npm i -S fuetify`

3. import

  ```javascript
  import {Button} from 'fuetify'
  import 'fuetify/dist/index.css'

  export default {
    name: 'app',
    components: {
      'f-button': Button
    }
  }
  ```
