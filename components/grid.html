f-row API

参数 说明 类型 默认值 必须 可选值
gutter 布局之间的间隙，单位是 px String 0 否 任意
align 布局内部的 flex 排列方式 String left 否 left|middle|right

f-col API

参数 说明 类型 默认值 必须 可选值
span 布局占几格 String 无 是 0-24
offset 布局左边的间隙占几格 String 0 否 0-24

TIP

布局为24分布局，span + offset 不能大于24
